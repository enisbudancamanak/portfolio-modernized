Toyroom

Game Dev
Unity
Blender

In this project, my girlfriend and I worked together to create a sandbox game. The basic idea was to give players a free experience without a fixed goal by allowing them to run around freely in the game world using an active ragdoll function. Another focus was a parkour element where players could interact with different objects in a child's room. The integration of a cannon made it possible to shoot the player in the desired direction using ragdoll physics, adding a dynamic dimension to the game.
